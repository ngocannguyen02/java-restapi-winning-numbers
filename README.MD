# Winning numbers

Winning numbers is a Java REST API with a mongo database.

## Installation

```bash
mvn clean install
docker build -t winning-numbers .
docker-compose up
```

## Usage

http://localhost:8080/swagger-ui.html#/jackpot-controller/
